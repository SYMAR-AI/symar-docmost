# SYMAR Knowledge Hub

The documentation and knowledge management platform for the SYMAR platform. Built on [Docmost](https://github.com/docmost/docmost), an open-source collaborative wiki.

## What this is

This is SYMAR's fork of Docmost, customized to serve as our internal knowledge base. We use it for team documentation, project knowledge, and operational guides.

## What we changed

- Rebranded to SYMAR Knowledge Hub
- Removed third-party branding from shared pages
- Custom deployment configuration (managed separately)
